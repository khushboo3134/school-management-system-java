# School Management System (Console Based)

## Description
A console-based School Management System built using Java.
The project follows layered architecture using model, service, and utility packages.

## Features
- User registration and login (file-based)
- Student management (Add, View, Search, Edit, Delete)
- Teacher management (Add, View, Search, Delete)
- Menu-driven console application
- File handling for data persistence

## Technologies Used
- Java
- OOP Concepts
- Collections (ArrayList)
- File Handling

## Project Structure
- model
- service
- util
- data
- Main.java

## How to Run
1. Clone the repository
2. Open the project in VS Code
3. Run Main.java
